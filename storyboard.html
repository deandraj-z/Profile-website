Storyboard

This page explains the design and structure of the website, including the purpose of each page and the overall theme.

List of Web Pages and Their Purpose

  • Index Page (index.html) - Introduces the user to the website with a welcoming message, a picture, and a navigation bar.
  • Professional Page (professional.html) - Showcases my professional background, resume, career goals, and vision/mission statements.
  • Personality Page (personality.html) - Highlights my personality traits, personal influences, and results from the Keirsey Temperament Sorter test.
  • Personal Development Page (development.html) - Reflects on my personal growth and lessons learned from the book Twelve Pillars.
  • Storyboard Page (storyboard.html) - Explains the design and structure of the website (this page).
  • Reference Page (reference.html) - Provides a list of all bibliographic sources used in the creation of the website.

Website Theme

The website uses a clean and professional design with a color palette of shades of gray, white, and blue for accents. The typography is based on the Roboto font, with responsive design and CSS animations for a modern look.

Target Audience

The website is designed for potential employers, academic peers, and anyone interested in learning more about my professional and personal background.
